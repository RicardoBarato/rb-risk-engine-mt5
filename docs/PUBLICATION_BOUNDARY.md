# Publication Boundary

This repository is the public educational version of `MT5 EA Auto Backtest Engine`.

It contains:

- educational MQL5 source code;
- public documentation;
- synthetic examples;
- public compile, automated backtest-loop and report-analysis tooling.

It does not contain:

- AI project code or analytics modules;
- non-public research artifacts;
- real broker reports or statements;
- real account identifiers;
- real `.set` production presets;
- non-public backtests or optimization outputs;
- proprietary strategy logic;
- internal planning documents.

Projects, reports, datasets, account details and research plans that are not explicitly safe for public release must stay outside this repository.
